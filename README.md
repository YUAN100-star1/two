鼠标点击器 (Auto Clicker)

一个用Python编写的双位置交替自动点击器。

项目简介

这是一个可以解放双手的自动化工具。它能够记录你在屏幕上选择的两点位置，然后在这两点间无限循环模拟鼠标点击，非常适用于重复性的UI操作测试、软件演示或自动化流程。

主要功能

双点循环：自动在用户指定的两个屏幕位置间循环点击。
热键控制：使用 F9 键一键启停程序，使用 ESC 键安全退出。
极速点击：默认点击间隔仅为95毫秒，效率高。
多线程设计：程序监听和点击动作互不干扰，响应灵敏。

环境依赖

在运行本项目之前，请确保你的Python环境中已安装以下库：

pip install pyautogui pynput
